# JavaScript Flash Cards

A simple and interactive JavaScript flash cards application for learning JavaScript concepts. This application helps users study and review key JavaScript concepts through an interactive card interface.

## Features

- Interactive flash cards with questions and answers
- Progress tracking showing current card position and completion percentage
- Navigation controls to move between cards
- Toggle between question and answer views
- 10 pre-loaded JavaScript concept questions 

## How to Use

1. Open `index.html` in your web browser
2. Click "Show Answer" to reveal the answer to the current question
3. Use the navigation buttons to move between cards:
   - "Previous" to go to the previous card
   - "Next" to advance to the next card
4. Track your progress with the progress bar at the top

## Technology Used

- HTML
- CSS
- Vanilla JavaScript
